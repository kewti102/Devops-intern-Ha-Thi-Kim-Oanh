ls: Lists files and directories in the current directory.
cd: Changes the current working directory.
pwd: Displays the path of the current directory.
mkdir: Creates a new directory.
rm: Removes files or directories.
cp: Copies files or directories.
mv: Moves or renames files and directories.
touch: Creates an empty file or updates a file's timestamp.
cat: Displays the contents of a file.
less: Views file contents page by page without opening an editor.
head: Displays the first lines of a file.
tail: Displays the last lines of a file.
grep: Searches for text matching a pattern.
find: Searches for files and directories.
xargs: Converts input into command-line arguments.
awk: Processes and analyzes text in columns and fields.
sed: Edits and transforms text line by line.
sort: Sorts lines of text data.
uniq: Removes or filters duplicate lines.
wc: Counts lines, words, or characters in a file.
tee: Writes output to both the screen and a file.
ps: Displays information about running processes.
top: Monitors system resources and processes in real time.
htop: Provides an interactive and user-friendly process viewer.
kill: Terminates a process using its PID.
nice: Runs a process with a specified priority.
df: Displays disk space usage.
du: Shows the size of files and directories.
free: Displays memory usage information.
uptime: Shows how long the system has been running.
uname: Displays system and kernel information.
who: Shows currently logged-in users.
chmod: Changes file or directory permissions.
chown: Changes file or directory ownership.
umask: Sets default file creation permissions.
tar: Archives multiple files into a single package.
gzip: Compresses files using the gzip format.
zip: Compresses files into a ZIP archive.
unzip: Extracts files from a ZIP archive.
ssh: Connects to a remote machine securely.
scp: Securely copies files between machines over SSH.
rsync: Synchronizes files and directories efficiently.
ln: Creates a hard link.
ln -s: Creates a symbolic (soft) link.
env: Displays environment variables.
export: Sets environment variables for child processes.
source: Reloads and executes commands from a shell script.
curl: Sends HTTP requests and transfers data from URLs.
wget: Downloads files from the internet.
which: Shows the path of an executable command.
whereis: Locates binaries, source files, and manual pages.
type: Identifies how a command is interpreted by the shell.
history: Displays previously executed commands.
alias: Creates command shortcuts.
echo: Prints text or variables to the terminal.
printf: Prints formatted output.
